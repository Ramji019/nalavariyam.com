Reminder Search

  • Type the reminder subject to be searched
  • Type the reminder details to be searched.
  • Type the date and Time of the reminder.
  • Click the search reminder button.
  • All the reminders based on your search will be displayed.
  • By right clicking on each reminder there are 3 menus.
  • View reminder menu is to view the reminder in detail.
  • Edit reminder is to edit the reminder and click save reminder
  • The modified reminder will be saved
  • Delete reminder is to delete the reminder if you no longer need it.
  • The + button near to each reminder will give you a brief description about the reminder.
