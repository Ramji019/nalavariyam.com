Outlet Search

  • Type the Appointment subject in the text field

  • Type the Appointment details in the text field.

  • Select the date and time of the appointment.

  • Click the Search Appointment button to search the appointment you entered.

  • It will display all the appointment details Appointment Subject, Appointment date, Appointment start time, Appointment End time.

  • Right clicking on each appointment a menu will be displayed.

  • View appointment is used to view the appointment details.

  • Edit appointment is used to edit the appointment.

  • While clicking edit appointment a small window will be opened where you can make the changes  and click the Add reminder Button.

  • The changes you have made will be updated.

  • Delete appointment is used to delete the Appointment.

  • Reset Button is to reset the values you entered.
